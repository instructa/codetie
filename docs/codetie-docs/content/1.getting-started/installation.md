## Installing codetie

For **existing** projects use:

```bash [Terminal]
npx codetie@latest init
```

To create a new project use

```bash [Terminal]
npx codetie@latest create
```

For local development inside a project:

```bash [Terminal]
npm install codetie --save-dev
# Then:
npx codetie@latest init
```
